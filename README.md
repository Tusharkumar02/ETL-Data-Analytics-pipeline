# 📊 End-to-End ETL Data Analytics Pipeline

## 🚀 Project Highlights
- Built an end-to-end ETL pipeline from raw Excel data to business insights  
- Cleaned and transformed data using Python (Pandas)  
- Performed analysis using SQL queries  
- Designed an interactive dashboard using Power BI  

---

## 📊 Business Problem
The objective of this project is to analyze sales data and generate insights related to:
- Sales growth trends  
- Product performance  
- Regional profitability  

---

## ⚙️ Tech Stack
- Python (Pandas, NumPy)  
- MySQL  
- Power BI  
- Microsoft Excel  

---

## 📊 Dashboard Insights
- 💰 Total Sales: **5M**  
- 💸 Total Profit: **1M**  
- 📈 Profit Margin: **26.05%**  

---

## 📈 Key Insights
- Printers generate highest revenue among all products  
- Sales peak during mid-year months  
- North region contributes highest profit
